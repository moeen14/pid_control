# PID Control Interactive Demo

This is an interactive web-based PID controller simulation designed for educational use.

## Features
- Tune Kp, Ki, Kd
- Step-changing setpoint (3 segments)
- Real-time visualization
- Performance metrics:
  - IAE
  - Overshoot Penalty
  - Control Effort
  - Total Cost (J)
  - Score

## How to Use
Open `index.html` in a browser or visit the hosted version.
